Git Questions and Answers

Q1. What is Git? / Git क्या है?

Ans: Git is a distributed version control system used to track changes in source code.

Q2: What is GitHub?

Ans: GitHub is a hosting platform for Git repositories with collaboration features.

Q3: What is the command to check Git version?

Ans: git --version

Q4: How to initialize a repository?

Ans: git init command is used to initialize a repository.

Q5: How to check current branch?

Ans: git branch
