Q6: What is a branch in Git?

Ans: A branch in Git is a separate line of development that allows you to work on different features or fixes independently.

Q7: How to create a new branch in Git?

Ans: Use git branch branch_name command.

Q8: How to switch between branches?

Ans: Use git checkout branch_name or git switch branch_name.

Q9: How to create and switch to a branch in one step?

Ans: Use git checkout -b branch_name.

Q10: How to merge a branch into the current branch?

Ans: Use git merge branch_name.

Q11: How to delete a branch?

Ans: Use git branch -d branch_name command.
